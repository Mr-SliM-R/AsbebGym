# Exercise Media

Drop licensed exercise animations here.

Place licensed files here and point an exercise to them through the database:

```text
/exercise-media/bench-press.gif
/exercise-media/push-ups.gif
/exercise-media/back-squat.gif
```

Store the media path in the exercise row using:

- `animation_media_type`: `image`, `gif`, `video`, `lottie`, or `three`
- `animation_src`: for example `/exercise-media/bench-press.mp4`
- `animation_credit`: optional source/license note

Use only files you have the rights to use.
